
# Premium Eid Cube 🎉

## Description

A 3D interactive Eid-themed cube built with React and Three.js. The cube rotates initially, then opens on click to reveal a glowing “Eid Mubarak” message with sparkling stars.

## Features

* Rotating 3D textured cube
* Click to open cube animation
* “Eid Mubarak” glowing text reveal
* Sparkling star effects
* Smooth animations and lighting

## How It Works

* Cube rotates at the start
* On click, cube faces open outward
* Text appears in the center with glow effect
* Stars animate around the scene



## Tech Used

* React
* Three.js
